Project Darkstar Server Core
The sgs-server module is the main subcomponent of the Project Darkstar server distribution. It depends on sgs-shared to complete its functionality. This site provides access to the source code repository, issue tracker, and mailing lists for this subcomponent. Please see http://www.projectdarkstar.com for more information.
Source code
The publicly accessible Subversion repository contains all of the source code for the sgs-server component. In addition to the java.net Subversion browser linked above, the repository may also be viewed using the FishEye browser.
Issue tracker
Issuezilla is used to track all issues with the sgs-server component including defects, enhancements, and outstanding tasks.
Mailing Lists
Subversion commit notifications and Issue tracker issue change notifications are sent to separate mailing lists for the sgs-server component. For development discussions and general announcements, please join the dev and announce lists for the games-darkstar parent project.
